# 🎮 Infinity Tic-Tac-Toe

각 플레이어가 최대 3개의 돌만 유지할 수 있는 혁신적인 틱택토 게임입니다. 4번째 돌을 두면 가장 오래된 돌이 자동으로 사라지는 FIFO(First In, First Out) 방식으로 무한한 전략적 플레이가 가능합니다.

## ✨ 주요 특징

### 🎯 핵심 게임 규칙
- **3개 제한 (FIFO)**: 각 플레이어는 최대 3개의 돌만 보드에 유지
- **무한 순환**: 4번째 돌을 두면 첫 번째 돌이 자동으로 제거됨
- **시각적 피드백**: 다음에 사라질 돌은 투명도와 깜빡임 효과로 표시
- **전략적 깊이**: 일반 틱택토와 달리 무승부가 거의 발생하지 않음

### 🤖 게임 모드
1. **PvP (2인 대결)**: 친구와 한 기기에서 번갈아가며 플레이
2. **PvE Easy**: 무작위로 착수하는 쉬운 AI
3. **PvE Hard**: 승리 기회를 포착하고 방어하는 전략적 AI

### 🎨 디자인
- **다크 모드**: 세련된 다크 배경
- **네온 컬러**: 파란색(Player 1) vs 핑크색(Player 2/AI)
- **애니메이션**: 부드러운 돌 배치/제거 효과
- **반응형**: 모바일, 태블릿, PC 모든 화면에 최적화

## 🎮 게임 플레이 가이드

### 기본 조작
1. **모드 선택**: 시작 화면에서 PvP 또는 PvE 모드 선택

## 🎯 주요 알고리즘

### FIFO Queue 관리
```javascript
// 예시: Player 1이 4번째 돌을 둘 때
if (player1Queue.length >= 3) {
    let oldPos = player1Queue.shift();  // 가장 오래된 위치 제거
    board[oldPos] = null;               // 보드에서 지우기
}
player1Queue.push(newPos);              // 새 위치 추가
```
